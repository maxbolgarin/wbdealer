# WBDealer

Telegram-бот для автоматического ведения канала с подборками товаров с Wildberries. Бот ищет товары, отбирает лучшие с помощью AI, собирает коллаж из фото и публикует пост в Telegram-канал по расписанию.

## Как работает

Бот запускает пайплайн по расписанию (по умолчанию в 9:00, 13:00 и 18:00 МСК):

1. **Выбор категории** — случайная категория из 8 доступных (кроссовки, электроника, одежда и т.д.), с ротацией чтобы не повторяться
2. **Поиск товаров** — запросы к WB Search API по ключевым словам категории, получение до 50 товаров на запрос
3. **Фильтрация** — отсечение по рейтингу, количеству отзывов, дедупликация, проверка что товар не публиковался ранее
4. **AI-курация** — LLM (через OpenRouter) выбирает 6 лучших товаров по соотношению цена/качество, скидке, разнообразию брендов
5. **Коллаж** — скачивание фото товаров с CDN Wildberries, сборка в сетку 3x2 (1080x1080) с округлёнными ячейками
6. **Публикация** — отправка поста с коллажем и описанием в Telegram-канал
7. **Сохранение** — запись в PostgreSQL для дедупликации и истории

## Формат поста

Каждый пост содержит:
- Коллаж-изображение из 6 фото товаров
- Заголовок с эмодзи категории
- Список товаров с кликабельными артикулами, эмодзи-рейтингом и ценой
- Легенду рейтинга (💰 хорошая цена, 👍 топ, 😱 дорого)

## Админ-команды

Бот принимает команды от администратора (по `ADMIN_USER_ID`):

| Команда | Описание |
|---------|----------|
| `/post` | Запустить генерацию поста вручную |
| `/status` | Статистика: количество постов, товаров, расписание |
| `/categories` | Список категорий с пометкой недавно использованных |

## Технологии

| Компонент | Технология |
|-----------|------------|
| Язык | Python 3.11+ |
| HTTP-клиент | httpx (async) |
| Telegram | aiogram 3.x |
| Коллажи | Pillow |
| LLM | OpenRouter (OpenAI-совместимый API) |
| База данных | PostgreSQL (asyncpg) |
| Конфигурация | pydantic-settings |
| Планировщик | APScheduler |
| Деплой | Docker + Docker Compose |

## Запуск

### 1. Подготовка

```bash
cp .env.example .env
```

Заполните `.env`:
- `TELEGRAM_BOT_TOKEN` — токен бота от [@BotFather](https://t.me/BotFather)
- `TELEGRAM_CHAT_ID` — ID канала (например `@your_channel`) или числовой ID
- `ADMIN_USER_ID` — ваш Telegram user ID (можно узнать у [@userinfobot](https://t.me/userinfobot))
- `OPENROUTER_API_KEY` — ключ API [OpenRouter](https://openrouter.ai/)
- `POSTGRES_PASSWORD` — пароль для PostgreSQL

### 2. Запуск через Docker Compose

```bash
docker-compose up --build -d
```

Это поднимет PostgreSQL и бота. Бот автоматически создаст таблицы при старте.

### 3. Проверка

Отправьте боту `/status` в личные сообщения — он ответит статистикой.
Отправьте `/post` для ручной генерации первого поста.

## Конфигурация

Все настройки через переменные окружения (файл `.env`):

| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
| `OPENROUTER_MODEL` | `anthropic/claude-sonnet-4` | Модель LLM на OpenRouter |
| `SCHEDULE_HOURS` | `[9,13,18]` | Часы публикации (МСК) |
| `PRODUCTS_PER_POST` | `6` | Товаров в одном посте |
| `MIN_RATING` | `4.0` | Минимальный рейтинг товара |
| `MIN_FEEDBACKS` | `10` | Минимум отзывов |
| `MAX_PRICE_RUB` | `10000` | Максимальная цена |
| `MIN_DISCOUNT_PCT` | `15` | Минимальная скидка (%) |

```
│   └── categories.py    # Справочник категорий

│   ├── curator.py       # AI-курация через OpenRouter

├── telegram/
│   ├── publisher.py     # Публикация в канал
│   └── admin.py         # Админ-команды бота
└── storage/
    └── db.py            # PostgreSQL (asyncpg)
```
